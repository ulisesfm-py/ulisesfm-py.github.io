Military Aircraft Classification with CNNs

Project Overview

The objective of this project was to classify images of military aircraft into their respective models using deep learning techniques. I compared two state-of-the-art convolutional neural network (CNN) architectures:

  • EfficientNetB3: Known for its efficiency and accuracy with fewer parameters.
  • ResNet50: A deep residual network that mitigates the vanishing gradient problem.

The dataset consisted of images categorized into various aircraft types, including fighters, bombers, transports, and unmanned aerial vehicles (UAVs). The goal was to build models capable of accurately identifying the aircraft model from an image, which has applications in defense analytics and aviation monitoring.

Data Preparation and Preprocessing

Key steps in data handling included:

  • Dataset Loading: Created functions to load images and labels from directories.
  • Data Reduction: Limited the number of samples per class to balance the dataset and reduce training time.
  • Image Augmentation: Applied transformations like horizontal flips to enhance model generalization.
  • Data Generators: Used TensorFlow's ImageDataGenerator for efficient batch processing.

I limited the maximum number of examples per label/class in the training dataset to 300. This was a decision made to reduce the computational time and avoid data imbalance, as there are some aircraft models that have lower example images in the training dataset.

Model Development and Training

EfficientNetB3 Model

The EfficientNetB3 model was chosen for its balance between accuracy and computational efficiency. Key features of the implementation included:

  • Preprocessing: Used the appropriate preprocessing function for EfficientNet.
  • Fine-tuning: Allowed the base model to be trainable to leverage pre-trained weights.
  • Regularization: Added dropout layers and regularizers to prevent overfitting.
  • Optimizer: Employed the Adamax optimizer with a custom learning rate schedule.

ResNet50 Model

The ResNet50 model was implemented to compare its performance with EfficientNetB3. Highlights of this model included:

  • Residual Blocks: Utilized skip connections to enable training of deeper networks.
  • Customization: Added fully connected layers and dropout for classification.
  • Preprocessing: Applied ResNet-specific preprocessing functions.

Custom Callback Function

To optimize training, I implemented a custom callback function that:

  • Dynamic Learning Rate Adjustment: Reduced the learning rate when the model plateaued.
  • Early Stopping: Halted training when no improvement was observed after a set number of epochs.
  • User Interaction: Allowed manual intervention to halt training based on performance metrics.

This approach ensured efficient use of computational resources and prevented overfitting.

Results and Evaluation

Both models were evaluated using the test dataset. The key metrics observed were:

EfficientNetB3 Model

  • Accuracy: Achieved an overall accuracy of 89%.
  • Precision, Recall, F1-Score: High scores across most classes, indicating reliable performance.

ResNet50 Model

  • Accuracy: Achieved an overall accuracy of 81%.
  • Performance: Slightly lower metrics compared to EfficientNetB3, but still demonstrated strong classification ability.

The confusion matrices for both models showed that misclassifications were rare, having a clear view of the diagonal components and a few missclassifications, most of the times corresponding to low quality images or images with strange perspectives.

Conclusion

The EfficientNetB3 model outperformed the ResNet50 model in this classification task, likely due to its optimized architecture for balancing depth and width of the network. The project demonstrated the effectiveness of modern CNN architectures in image classification tasks involving multiple classes with subtle differences.

Future Work

Future enhancements could include:

  • Data Expansion: Collecting more images to improve model generalization.
  • Ensemble Methods: Combining predictions from multiple models to boost accuracy.
  • Advanced Augmentation: Using techniques like Mixup or Cutout to further augment the training data.
